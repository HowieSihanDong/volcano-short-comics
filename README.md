# 火山短漫 - Volcano Short Comics

一个展示前端顶尖技术的企业级网站，采用 Apple 风格设计语言，具有流畅的动画效果和现代化的交互体验。

## 🌟 特性

- **Apple 风格设计** - 深色主题、玻璃拟态效果、精致的排版
- **流畅动画** - GSAP + ScrollTrigger 驱动的滚动动画
- **水平滚动展示** - 作品展示区域采用水平滚动效果
- **自定义光标** - 磁性悬停效果的自定义光标
- **响应式设计** - 完美适配桌面和移动设备
- **加载动画** - SVG 动画加载屏幕
- **全屏菜单** - 优雅的导航菜单动画

## 🛠 技术栈

- **框架**: Vite 5.4
- **动画**: GSAP + ScrollTrigger
- **平滑滚动**: Lenis (已禁用，使用原生滚动)
- **样式**: 原生 CSS (CSS Variables)
- **字体**: Inter (Google Fonts)

## 📁 项目结构

```
├── index.html          # 主页面
├── src/
│   ├── main.js         # 主要 JavaScript 逻辑
│   └── style.css       # 样式文件
├── dist/               # 构建输出
└── package.json
```

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 📝 主要功能

### 页面区块
1. **Hero 区域** - 3D 形状动画、文字揭示效果
2. **作品展示** - 水平滚动展示精选作品
3. **关于我们** - 团队介绍和特色功能
4. **服务** - 卡片式服务展示
5. **联系** - 表单和社交媒体链接

### 动画效果
- 文字逐行/逐字揭示
- 视差滚动
- 磁性按钮效果
- 卡片悬停缩放
- 进度条指示器
